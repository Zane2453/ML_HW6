## Python Virtual Environment
```
python3 -m venv /path/to/venv/dir       # create venv
source /path/to/venv/dir/bin/activate   # activate venv
```

## Install dependencies
```
pip3 install -r requirement.txt
```

## The Code Usage
```
python3 main.py -h
```

## Start
```
python3 main.py
```
